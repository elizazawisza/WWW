Trylma/Chińskie warcaby

Implementacja popularnej gry "chińskie warcaby". Użytkownik może grać w grę z botami lub z innymi użytkownikami poprzez połączenie się do prostego serwera.
Użytkownik może tworzyć grę lub dołączyć do obecnie trwającej gry, w momencie, gdy rozgrywka jeszcze się nie rozpoczęła.
Po uruchomieniu programu pokazuje się menu, w którym można wybrać ilu graczy ludzi oraz ile botów będzie brało udział w rozgrywce.

Najciekawszy fragment kodu odpowiadający za przyłączanie graczy do gry

        
    public void runGameServer(int noPlayers,int noBots){
         stillInGame = false;
         GameServer game = new GameServer(this, noBots, noPlayers);
         try {
             if(serverSocket == null)
                  serverSocket = new ServerSocket(myPort);
             System.out.println("Start server.");
         } catch (IOException ex) {
             System.out.println(ex.toString());
             System.exit(-1);
         }
         for(int i = 1; i <= noPlayers; i++){
             try {
                 PlayerInstance player = new HumanPlayer(game, i, serverSocket.accept());
                 game.addPlayer(player);
                 player.start();
                 System.out.println(i);
             } catch (IOException e) {
                 e.printStackTrace();
                 continue;
             }
         }
     }
